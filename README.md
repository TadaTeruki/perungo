# ルールと操作

矢印キー(or WASD): 移動する・ブロックを押す
Enterキー: ブロックを壊す

ブロックを押して敵を倒せます。一気に倒せば高得点。
敵を全て倒したらクリアです！

# 実行方法

```
$ rye sync
$ rye run python -m perungo
```
